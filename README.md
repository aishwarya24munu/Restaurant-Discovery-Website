Restaurant Discovery Website (Zomato Inspired UI):-

A restaurant discovery website inspired by Zomato, developed using HTML, CSS, and JavaScript.
This project focuses on creating a clean and interactive frontend interface while understanding how modern web applications are designed. The goal of this project was to improve frontend development skills by building a real-world inspired user interface.

 Features->

- Modern restaurant discovery homepage
- Location and restaurant search interface
- Dining, Delivery, and Nightlife sections
- Restaurant collections section
- Promotional offer banner
- Filter options
- Restaurant listing cards with ratings and details
- Interactive tab switching
- Hover effects and smooth UI experience


 Tech Stack->
 
- HTML5
- CSS3
- JavaScript


Concepts Practiced->

- Structuring web pages using HTML
- Styling and layouts using CSS
- CSS Flexbox
- CSS Grid
- Responsive design concepts
- JavaScript DOM manipulation
- Event handling
- Creating reusable UI components

 Project Objective->

The main objective of this project was to strengthen frontend development skills by recreating a real-world style restaurant platform interface.
Through this project, I learned how to organize website sections, create attractive layouts, handle user interactions, and improve the overall user experience.
